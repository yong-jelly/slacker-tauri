# Tauri 앱 아이콘 관리 가이드

Tauri 프로젝트에서 앱 아이콘을 설정하고 관리하는 방법에 대한 상세 내용입니다.

## 1. 아이콘 생성 도구 (Tauri Icon CLI)

Tauri는 하나의 소스 이미지(SVG 또는 고해상도 PNG)를 기반으로 모든 플랫폼에 필요한 아이콘들을 자동으로 생성해주는 CLI 도구를 제공합니다.

### 사용 방법
1. 소스 이미지 준비 (가급적 1024x1024 이상의 SVG 권장)
2. 아래 명령어를 터미널에서 실행:

```bash
# npm 사용 시
npm run tauri icon path/to/app-icon.svg

# bun 사용 시
bun tauri icon path/to/app-icon.svg
```

### 생성되는 파일들
명령어 실행 시 `src-tauri/icons` 디렉터리에 다음 파일들이 생성/교체됩니다:
- `32x32.png`: 리눅스 및 기타 용도
- `128x128.png` / `128x128@2x.png`: 일반적인 앱 아이콘
- `icon.icns`: macOS용 아이콘 묶음
- `icon.ico`: Windows용 아이콘 묶음
- 기타 다양한 크기의 PNG 파일들 (Windows 스토어 등 대응)

## 2. 프로젝트 설정 (`tauri.conf.json`)

생성된 아이콘들은 `src-tauri/tauri.conf.json`의 `bundle > icon` 섹션에 정의되어야 합니다.

```json
"bundle": {
  "icon": [
    "icons/32x32.png",
    "icons/128x128.png",
    "icons/128x128@2x.png",
    "icons/icon.icns",
    "icons/icon.ico"
  ]
}
```

## 3. Flow 스타일 아이콘 제작 팁 (SVG)

Flow 앱과 같은 미니멀한 스타일을 유지하려면 아래와 같은 SVG 구조를 활용할 수 있습니다.

```svg
<svg width="1024" height="1024" viewBox="0 0 1024 1024" fill="none" xmlns="http://www.w3.org/2000/svg">
  <!-- 배경: Flow 특유의 부드러운 색상 (#FF5F5F 등) -->
  <rect width="1024" height="1024" rx="220" fill="#FF5F5F"/> 
  <!-- 미니멀한 타이머 또는 로고 형상 -->
  <circle cx="512" cy="512" r="320" stroke="white" stroke-width="60"/>
  <path d="M512 320V512L640 640" stroke="white" stroke-width="60" stroke-linecap="round" stroke-linejoin="round"/>
</svg>
```

## 4. 주의사항
- **해상도**: 소스 이미지는 최소 1024x1024 픽셀 이상이어야 macOS의 Retina 디스플레이에서 깨지지 않습니다.
- **포맷**: SVG 사용 시 텍스트는 반드시 `Outline` 처리(Path로 변환)해야 폰트 유실 문제를 방지할 수 있습니다.
- **투명도**: macOS 아이콘은 원형이나 특정 모양 외곽에 투명도가 포함될 수 있지만, Windows 아이콘은 가급적 꽉 찬 사각형 형태가 권장됩니다.
