[2025/12/23]
위젯 모드 전환과 macOS 메뉴바 트레이 표시 개선 😘
- 여러 작업이 동시에 실행 중일 때 마지막에 실행된 작업의 정보가 메뉴바에 표시되도록 개선
- 작업이 일시정지되어도 다른 실행 중인 작업이 있으면 마지막으로 실행된 작업의 이름과 남은 시간 유지
- 모든 작업이 일시정지되거나 완료되면 메뉴바에 기본 앱 이름 표시

[다중 모니터 환경 지원 강화]
- 위젯 모드에서 일반 모드로 전환 시 창이 화면 밖으로 벗어나지 않도록 자동 조정
- 모니터 변경이나 제거 시에도 창이 화면 밖에 표시되지 않도록 보완
- 위젯 모드에서도 현재 창 위치 유지, 사용자가 직접 조정 가능

[위젯 모드 크기 및 DPI 스케일링 개선]
- 위젯 모드 전환 시 크기가 300x300으로 자동 변경되도록 개선
- DPI 변환 오류 수정으로 창 크기 정상 유지

[위젯 모드 창 크기 변경 권한 추가]
- 위젯 모드와 일반 모드 간 전환 시 올바른 크기 및 위치 복원을 위한 권한 추가
- 추가된 권한
 - core:window:allow-set-size : 창 크기 변경
 - core:window:allow-set-position : 창 위치 변경
 - core:window:allow-set-resizable : 리사이즈 가능 여부 변경
